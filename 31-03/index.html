O que é html

A respeito da linguagem de marcação

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar a estrutura de páginas web. Ele utiliza tags para organizar o conteúdo — como títulos, parágrafos, imagens e links — formando o esqueleto do site, que é interpretado pelos navegadores.
